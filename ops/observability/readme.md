# Prometheus/Grafana dashboards, Loki and Tempo for observability
# # Ce fichier docker-compose.yml permet de lancer plusieurs services pour l'observabilité
# # Pour lancer les services, utilisez la commande suivante :
# # docker-compose up -d
# # docker-compose.yml
# # Pour arrêter les services, utilisez la commande suivante :
# # docker-compose down
# # Pour accéder à Grafana, ouvrez votre navigateur et allez à l'adresse suivante :
# # http://localhost:3000
# # Pour accéder à Loki, ouvrez votre navigateur et allez à l'adresse suivante :
# # http://localhost:3100
# # Pour accéder à Tempo, ouvrez votre navigateur et allez à l'adresse suivante:
# # http://localhost:3200
